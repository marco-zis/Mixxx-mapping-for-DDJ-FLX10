# Mixxx-mapping-for-DDJ-FLX10
Mixxx XML and JS mapping for the Pioneer controller DDJ-FLX10

Mixxx is a great piece of software, and for the art of DJing it is usefull to use a controller. These need a series of code to interact properly with software, called the mapping. The mapping basically affects functions of the software to the faders, buttons, knobs, etc of the controler.

I started my journey into DJing with Mixxx and the DDJ-FLX4, but later added the DDJ-FLX10 to my gear, and realised there was no mapping available. I found a mapping for the DDJ-1000 that I started to adapt. This first version provides the basic functions to use the FLX10 with Mixxx, but needs testing to evolve. As the DDJ-FLX10 has a lot of features, there is room for much improvement to progressivelly create a magic duo : the DDJ-FL10 completely working with Mixxx.
